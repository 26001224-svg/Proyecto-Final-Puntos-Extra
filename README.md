# Proyecto-Final-Puntos-Extra
Puntos extras 
Funcionamiento del programa

El programa permite realizar cotizaciones de seguros de forma automática. Primero solicita la edad del asegurado y verifica que sea mayor de edad. Si cumple con la condición, calcula el costo del seguro a partir de un precio base y le agrega recargos según la edad, si está casado, la cantidad de hijos, las propiedades y el salario. Al final muestra el total a pagar.

Partes del problema que se consideran

El programa considera que el asegurado sea mayor de edad, aplica recargos según la edad, toma en cuenta si está casado y la edad del cónyuge, la cantidad de hijos, la cantidad de propiedades, los ingresos del asegurado y permite realizar varias cotizaciones sin reiniciar el programa.

Posibles mejoras

Se podría mejorar validando mejor los datos ingresados, ordenando el código con funciones, mejorando la interacción con el usuario y guardando un historial de cotizaciones.
